# Project-ArchiAkuta

A public repository to hold userscripts and related assets for use with [Violentmonkey](https://violentmonkey.github.io/).

## About

This repo serves as a hosting location for userscripts that can be installed via Violentmonkey, a browser extension for managing and running userscripts.

## Usage

1. Install the [Violentmonkey](https://violentmonkey.github.io/) extension for your browser.
2. Navigate to any `.user.js` script file in this repository.
3. Click the **Raw** button to view the raw file content.
4. Violentmonkey should automatically detect the userscript and prompt you to install it.
